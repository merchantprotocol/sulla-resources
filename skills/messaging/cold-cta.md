# Cold Call to Action

This skill crafts a cold call-to-action for lukewarm audiences. It mixes and matches elements from the empathy map into a quick, punchy CTA.

---

## Input

Read:
- `~/sulla/identity/business/business-owner-perspective-empathy-map.md`
- `~/sulla/identity/business/onboarding.md`
- `~/sulla/identity/business/brand-script.md` (if available — reuse the bribe)

---

## Cold CTA Formula

```
At (company) we know you want (desire). Get started (Bribe), and (Product). To stop (problem/pain) and instead (have the desire).
```

This is a compressed version of the brand script — just the opening desire, the entry point, the product, and the contrast between pain and desire.

---

## Process

1. Pull company name, desire, bribe/free entry, product, problem/pain from the empathy map and brand script.
2. Generate **3-5 variations**:
   - Vary which pain element you use (external problem vs. emotional pain vs. fear)
   - Vary the CTA intensity (soft ask vs. direct ask)
   - Try different lengths (one-liner vs. two sentences)
3. Recommend the best version for each context:
   - Website banner / hero CTA
   - Social media bio / post CTA
   - Ad copy CTA
   - Email signature CTA

## Rules

1. **Keep it short.** This is a CTA, not a story. The brand script handles the long version.
2. **The bribe lowers the barrier.** Always include the free entry point if one exists.
3. **Contrast is the hook.** "Stop [pain] and instead [desire]" — this contrast is what drives action.
4. **Match the context.** A website hero CTA can be longer than a social bio CTA.
