# Kubernetes Tools

Manage Kubernetes resources.

## kubectl_apply
```bash
sulla kubectl/apply '{"filePath":"/path/to/manifest.yaml"}'
```

## kubectl_describe
```bash
sulla kubectl/describe '{"resource":"pod","name":"my-pod","namespace":"default"}'
```

## kubectl_delete
```bash
sulla kubectl/delete '{"resource":"pod","name":"my-pod","namespace":"default"}'
```
