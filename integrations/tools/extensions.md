# Extensions Tools

Manage Sulla marketplace extensions.

## list_extension_catalog
```bash
sulla extensions/list_extension_catalog '{}'
```
List all available extensions from the marketplace.

## list_installed_extensions
```bash
sulla extensions/list_installed_extensions '{}'
```
List currently installed extensions with versions and upgrade status.

## install_extension
```bash
sulla extensions/install_extension '{"extensionId":"ext-123"}'
```

## uninstall_extension
```bash
sulla extensions/uninstall_extension '{"extensionId":"ext-123"}'
```
